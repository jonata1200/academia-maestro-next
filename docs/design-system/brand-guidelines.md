# Brand Guidelines (inicial)

## Identidade (atual)
- **Cor primária**: azul `#0000FF`
- **Cor secundária**: branco `#FFFFFF`
- **Texto**: `#333` (principal) e `#555` (secundário)
- **Cor escura**: `#000033`
- **Fonte**: Poppins

## Direções
- Priorizar contraste alto para leitura.
- Usar a cor primária para CTAs e destaques (evitar uso excessivo).
- Manter espaçamento consistente (escala de spacing na Fase 2).
