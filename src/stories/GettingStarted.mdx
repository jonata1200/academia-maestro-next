# Getting Started

## Como importar

Tokens:
- `import { tokens } from '../design-system/tokens'`

Componentes:
- `import { Button, Input } from '../design-system/components'`

Hooks:
- `import { useModal, ToastProvider } from '../design-system/hooks'`

## Convenções
- Preferir **composição** de componentes base.
- Evitar valores hardcoded (cores/espaçamento) fora dos tokens.
- Acessibilidade por padrão: teclado, foco visível e semântica.
