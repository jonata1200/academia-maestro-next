# Tokens

Os **tokens** são a base do Design System da Academia Maestro. Eles são valores de design reutilizáveis que garantem consistência visual em toda a aplicação.

## O que são Tokens?

Tokens são variáveis que representam decisões de design fundamentais:
- **Cores**: Paleta de cores da marca e semânticas
- **Tipografia**: Famílias de fonte, tamanhos e pesos
- **Espaçamento**: Escala consistente de espaçamentos
- **Animações**: Durações e curvas de animação
- **Breakpoints**: Pontos de quebra responsivos
- **Sombras**: Elevação e profundidade
- **Bordas**: Raio de borda e estilos

## Tipografia

### Família de Fonte

Fonte padrão: **{typography.fontFamily.sans.join(', ')}**

### Tamanhos de Fonte

O sistema utiliza uma escala tipográfica que varia de `xs` (0.75rem) até `5xl` (3rem).

Exemplo de uso:
```typescript
import { typography } from '@design-system/tokens';

// Aplicar tamanho base
className={typography.fontSize.base[0]}
```

### Pesos de Fonte

- Light (300)
- Regular (400)
- Medium (500)
- Semibold (600)
- Bold (700)

## Espaçamento

O sistema utiliza uma escala de espaçamento baseada em rem (root em), garantindo consistência e responsividade.

| 2 | {spacing[2]} | 8px | Espaçamento pequeno |
| 3 | {spacing[3]} | 12px | Espaçamento médio-pequeno |
| 4 | {spacing[4]} | 16px | Espaçamento padrão |
| 6 | {spacing[6]} | 24px | Espaçamento médio |
| 8 | {spacing[8]} | 32px | Espaçamento grande |
| 12 | {spacing[12]} | 48px | Espaçamento extra grande |
| 16 | {spacing[16]} | 64px | Espaçamento máximo |

## Como usar Tokens

### Importar tokens

```typescript
import { colors, typography, spacing } from '../design-system/tokens';

// Usar cores
<div style={{ backgroundColor: colors.brand.primary[500] }}>

// Usar espaçamento
<div style={{ padding: spacing[4] }}>

// Usar tipografia
<p style={{ fontFamily: typography.fontFamily.sans.join(', ') }}>
```

### Com Tailwind CSS

Os tokens também são disponibilizados via Tailwind CSS através das configurações do projeto, permitindo uso como classes utilitárias.

## Boas Práticas

- ✅ **Use sempre tokens** em vez de valores hardcoded
- ✅ **Mantenha consistência** usando a mesma escala de espaçamento
- ✅ **Respeite a paleta** de cores definida
- ❌ **Não crie** novos valores de espaçamento fora da escala
- ❌ **Não use** cores que não estejam nos tokens
